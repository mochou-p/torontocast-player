# torontocast player
terminal-based internet radio player for torontocast stations

## usage
build with `make`/`make debug`  
run with `make run`/`make run_debug`

## requirements
|                    |                               |
|-------------------:|:------------------------------|
|                 OS | Linux                         |
|              shell | bash                          |
|   runtime binaries | curl, ffplay, grep, img2sixel |
| build dependencies | gcc, make                     |
